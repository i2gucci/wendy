Process

How It Works

Simple four-step process from installation to order completion.

1

Install Dependencies

Install Python packages and Playwright browser drivers with a single command. All dependencies managed through pip for seamless setup.

2

Configure Your Order

Define your delivery method, location, and menu items using the intuitive OrderBuilder API or pre-configured combo templates.

3

Execute the Bot

Run the script and monitor real-time progress as the bot navigates Wendy's website, adds items, and proceeds to checkout automatically.

4

Complete Payment

Review your order details and manually complete the secure payment process. The bot handles everything up to checkout for safety.

Documentation

Quick Start Guide

Get up and running in minutes with our simple API.

example.py 
# Install dependencies
pip install -r requirements.txt
playwright install chromium

# Import the bot and builder
from wendys_bot import WendysBot
from order_config import OrderBuilder

# Build your order with method chaining
order = (OrderBuilder()
    .set_delivery_method('pickup')
    .set_location('10001')
    .add_item("Baconator")
    .add_item("Large Fries")
    .add_item("Medium Frosty")
    .build()
)

# Initialize and execute
bot = WendysBot(headless=False, slow_mo=1500)
bot.place_order(order)
